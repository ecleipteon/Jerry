FraWS 0.1 è in fase di testing


Abstract:
L'obiettivo del progetto è presentare un'applicazione web server che implementi il protocollo HTTP e che riesca a gestire al meglio un gran numero di connessioni. Durante la progettazione dell'applicativo è stato deciso di utilizzare il linguaggio C e le API da questo fornite per la creazione di una connessione affidabile sfruttando il protocollo TCP, come previsto dalle specifiche di HTTP. Due sono stati, fino a questo momento, gli aspetti più impegnativi affrontati: la gestione del maggior numero di connessioni possibile (problema anche noto come C10K problem) e il parsing delle richieste HTTP. L'applicazione presentata non è ancora completa, molti aspetti applicativi andranno implementati successivamente.


L'applicazione è stata sviluppata in linguaggio C Luca Maria Castiglione.
Il codice, rilasciato sotto licensa GNU/GPL, sarà presto disponibile online.
